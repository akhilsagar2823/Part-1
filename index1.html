Git Commands

  • git init - to initialize git repository
  • touch (fn) - to creat a file
  • git status - to check repository status
  • git add (fn) - to track\stage file
  • git commit - to commit a file
  • git add -A - to track\stage all the filess
  • git commit -m"(commit message)" - to commit a file
  • git checkout (fn) - to bring previous commited version of file
  • git checkout -f - to checkout all files
